# Genesis Coaching Access

This is the access desk for HeroForge Genesis coaching materials.

If you're a current Genesis client, request access here:

## [Request access](https://github.com/MAMDV/genesis-access/issues/new?template=access-request.yml)

What happens next:

1. Mark approves your request personally, usually within a day.
2. GitHub emails you an invitation to the private Genesis library.
3. Click "Accept invitation" in that email. The invitation expires after 7 days.

Not a Genesis client? Requests from non-clients are declined.
